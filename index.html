Simple Tagging Example

If your tagging box is part of a form, you may want to specify select: true. Tagit will then store all tags values in a hidden <select> element. Each <option> element will contain the URI encoded tag name. In this example, and all examples that follow, tag delimiters are comma, enter, space, semicolon, and tab. These are configurable. See the triggerKeys option. Also note that you can paste tags in, and they will be delimited using any declared printable trigger keys. Tags can be removed by clicking on the 'x' icon, or simply backspacing (twice).

    Autocompleter Example

    Tagit makes use of jQuery UI Autocomplete. Simply specify an array of values to make available to the autocompleter via the tagSource option. In this example, strings 'one' through 'six' are available for autocompletion.


      "Sortable" Example

      Tagit also makes use of jQuey UI Sortable. Simply specify sortable: true when initializing your tagging box to allow users to rearrange tags by clicking and dragging. Specify sortable: 'handle' to include a drag handle on each tag. This example includes drag handles. Without drag handles, the entire tag is draggable.


        Edit Existing Tags Example

        To edit an existing tag, simply specify editOnClick: true during initialization. You may then edit a tag simply by clicking on it.

          There are many additional optional configuration points for the Tagit library. Please refer to the project README for details.
